# 纸牌游戏计分器 - 微信小程序

一个多人实时联机计分工具，采用表格形式展示，适合桌游、棋牌、麻将、剧本杀、羽毛球、乒乓球等需要多局、多人、逐局记录分数的场景。

## 核心功能

### 🏠 房间系统
- 任何人可创建房间，生成 6 位数字房间号
- 其他人输入房间号即可加入同一局计分
- 一个房间支持 2~8 人自由增减

### 📊 表格式计分板（核心界面）
```
┌─────────┬─────────┬─────────┬─────────┐
│  玩家   │  玩家A  │  玩家B  │  玩家C  │
├─────────┼─────────┼─────────┼─────────┤
│  第1局  │    10   │    20   │    15   │
├─────────┼─────────┼─────────┼─────────┤
│  第2局  │    15   │    25   │    30   │
├─────────┼─────────┼─────────┼─────────┤
│  总分   │   +25   │   +45   │   +45   │
└─────────┴─────────┴─────────┴─────────┘
```

- **第一行**：玩家名称（可点击修改）
- **中间行**：第几局（第1局、第2局…可无限添加）
- **最后一行**：自动计算总分（求和，无需手动算）
- **每一格分数**：点击即可输入数字，实时保存并同步

### 🔄 实时同步
- 改名字 → 所有人同步
- 改分数 → 所有人同步
- 加局数 → 所有人同步
- 加玩家 → 所有人同步

### ✨ 操作简单
- 点击格子输入数字
- 点击名字修改昵称
- 一键添加新一局
- 一键添加新玩家
- 一键重置/清空

## 技术栈

- 微信小程序原生开发 (WXML/WXSS/JavaScript)
- 微信云开发 (Cloud Base)

## 数据结构

```javascript
{
  roomId: "123456",           // 6位数字房间号
  roomName: "今晚斗地主",      // 房间名称
  creatorId: "用户openId",     // 创建者ID
  maxPlayers: 8,              // 最大玩家数
  players: [                  // 玩家列表（列）
    { id: "p1", name: "玩家A", userId: "xxx", isCreator: true },
    { id: "p2", name: "玩家B", userId: "xxx", isCreator: false }
  ],
  scores: [                   // 分数二维表 [行][列]
    [10, 20],                 // 第1局
    [15, 25],                 // 第2局
    [5, 30]                   // 第3局
  ]
  // 总分自动计算：每个玩家所有局分数相加
}
```

## 部署指南

### 1. 准备工作
1. 注册微信小程序账号
2. 下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
3. 获取小程序 AppID

### 2. 配置项目
1. 使用微信开发者工具打开项目
2. 修改 `project.config.json` 中的 `appid` 为你的小程序 AppID
3. 开通微信云开发：
   - 在微信开发者工具中点击「云开发」按钮
   - 按照指引开通云开发环境
   - 创建云数据库集合：`rooms`

### 3. 配置数据库权限
在云开发控制台，为 `rooms` 集合配置权限：
- 所有用户可读
- 仅创建者可读写

### 4. 上传云函数
在微信开发者工具中：
1. 右键点击 `cloudfunctions` 目录
2. 选择「当前环境」-> 选择你的云开发环境
3. 右键点击每个云函数目录，选择「上传并部署：云端安装依赖」

需要上传的云函数：
- createRoom
- joinRoom
- addPlayer
- deletePlayer
- addRound
- updateScore
- updatePlayerName
- resetScores

### 5. 编译预览
1. 点击微信开发者工具的「编译」按钮
2. 在模拟器中测试小程序功能

### 6. 发布上线
1. 点击「上传」按钮上传代码
2. 登录 [微信小程序管理后台](https://mp.weixin.qq.com)
3. 提交审核并发布

## 使用说明

### 创建房间
1. 打开小程序，授权用户信息
2. 点击「创建房间」
3. 输入房间名称，选择最大玩家数
4. 点击「创建房间」，获得6位邀请码
5. 分享邀请码给好友

### 加入房间
1. 打开小程序，授权用户信息
2. 点击「加入房间」
3. 输入好友分享的6位邀请码
4. 点击「加入房间」

### 计分操作
1. **修改分数**：点击表格中任意分数格子，输入数字后确认
2. **修改玩家名称**：点击顶部玩家名称，输入新名称后确认
3. **添加一局**：点击底部「添加一局」按钮
4. **添加玩家**：房主点击底部「添加玩家」按钮
5. **重置分数**：房主点击底部「重置分数」按钮

### 其他功能
- **复制房间号**：点击顶部的房间号区域
- **分享房间**：点击右上角分享按钮
- **离开房间**：点击右上角退出按钮

## 注意事项

1. 云开发有免费额度限制，超出后需要付费
2. 房间数据会保存在云端，关闭小程序后数据不会丢失
3. 只有房主可以添加/删除玩家和重置分数
4. 建议定期清理不用的房间以节省存储空间
5. 分数可以是正数或负数
